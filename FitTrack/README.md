# FitTrack

## Setup Instructions

1. Clone the repository.
2. Navigate to each service directory and install dependencies using `npm install` or `pip install -r requirements.txt` for Python services.
3. Set up environment variables by copying `.env.example` to `.env` and filling in the required values.
4. Start the services using Docker Compose: `docker-compose up --build`.
5. Access the frontend at `http://localhost:3000`.
